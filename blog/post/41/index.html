既然要写 2025，那就干脆把前面的经历一并写了吧，当作是对自己前半生的一次回顾。

我是如何走上前端这条路的

最早的“编程”接触（小学 → 初中） 💻

大概小学二年级，七岁左右，家里买了第一台电脑。
那时候天天玩《植物大战僵尸》🌱🧟‍♂️。

当年的配置我还记得个大概：

  • AMD A6 APU（具体哪款忘了，好像是 3 开头）
  • DDR3 4G
  • 500G 机械硬盘

后来开始玩《穿越火线》，再到五、六年级，沉迷《造梦西游》。
但我嫌刷装备太麻烦，于是加了不少 QQ 群，问别人怎么搞装备 🤔。

第一次真正“接触技术”，就是那会儿——
有人教我用 CE，搜白装的装备名开头，改成紫金装备。
捡起来之后居然一直存在，直接数值碾压一路秒怪 ⚡。

现在回头看，这大概就是我第一次和“编程 / 逆向”沾边。

初中：逃离现实，彻底沉进网络 🕳️

上了初中之后，家里做生意，基本没人管我。
再加上在学校遭遇了很严重的霸凌，初二基本已经放弃学习了。 初一学习成绩年级前100，90多名吧。班级是30多名左右。(因为在2班当时是按照入学考试成绩分班的。(先分班，考试成绩再分班)). 然后就三年都在一个班级里了。

那时候在 牛马一中 念书：

  • 初二时因为和班主任侄子起冲突（他长期霸凌我），我把他打进了医院
  • 班主任闹着要开除我
  • 最后是我妈托远房亲戚（副校长）才保下来
  • 之后我只能蹲在走廊外听课 🚪

那段时间状态很糟，一度玉玉，还做过很傻的事，后来被迫停课在家两天。
但最后还是被家里强行拉回学校继续读。

学校是住宿制，现实里几乎没朋友。
只有一个同样被霸凌的同学算朋友，不过他后来还是持续被欺负。

于是，我几乎把整个精神世界都放进了网络 🌐。

易语言、外挂、第一次“靠技术赚钱” 💰

因为游戏菜，就开始研究外挂。 那时候接触到了 易语言和x86汇编。

周末就干这些事：

  • CE + OD 找基址
  • 用易语言封装界面
  • 在 真牛论坛 / 精益论坛 发外挂

甚至还：

  • 做了外挂官网吗
  • 备案了 🤣

后来靠卖挂的钱，加盟了卡盟，当代理站，在自己 QQ 群里卖卡。
对一个初中生来说，几千块真的不少。

做的是 QQ群校验 才能用挂，
钱打到支付宝，留言 QQ 拉群。

靠自己赚的钱：

  • 买了第一部手机（小米 3 📱）
  • 换了电脑显卡
  • CF 买了不少神器 🔫

现在回头看，三观很有问题，但那是我第一次意识到：技术能换钱。

2017 ~ 2018：中专，两年精神消耗 🪖

中考 360 多分，刚过普高线。
但学费一年 2~3 万，家里扛不住，读了一周就被拉去读中专。

后来我才知道所谓的“授渔计划”：

  • 每年给家长几千
  • 学费一年 800
  • 每周 120 生活费

中专是军事化管理：

  • 天天教官好、老师好
  • 喊口号
  • 说实话，精神折磨程度堪比衡水 😵‍💫

专业是 汽修，我完全不感兴趣，也什么都没学会。
家里给我的理由是：

“在本地修车，一个月五六千，很高了。”

没有电脑，一个月才回家两天。
两年时间，基本就是在宿舍打游戏混过去的 🎮。

2019：辍学、电子厂、自学编程 🔧📚

最终还是受不了那种生活，决定不读了。
自己买火车票，跑去 武汉电子厂拧螺丝。

白天拧螺丝，晚上学：

  • HTML
  • CSS
  • PHP

那时候真的要感谢 向军老师 / 后盾人 🙏。
B 站当年还没什么免费课，黑马之类都是 20 年后才开始放。

学了大概 3 个月，就离职去找 PHP 相关的工作。

高考 & 大专（走个形式） 🎓

因为没学历，一时脑热去参加了高考。
喜提一个 大专 —— 150 分就能录。

本来报的软件工程，分不够，
最后被调剂到 移动通信（2 年制）：

  • 修电脑
  • 弱电
  • 水晶头
    23333……

2020：大学半年，直接实习跑路 🏃‍♂️

开学后贷款读大学。
因为各种原因和同学格格不入，只读了半年。

和辅导员商量，提前出去实习。
别人第二年才实习，我直接跑路。

2021 ~ 2023：外包、疫情、前端转型 🌪️

苏州 & 疫情

2021 年来到苏州，因为朋友在这。
结果一来就被疫情困了三个月。

原计划是：

  • 先找 PHP 工作
  • 私下学 Vue，转前端

但现实是：

  • PHP 行情极差
  • 基本只剩模板站、TP 接口、私活

第一家公司月薪 4k，
而且：

  • 天天 10 点下班
  • 没事也不让走
  • 要陪销售打电话 📞

那家公司后来倒闭了。
简历我都不太敢写。

全职学前端 → 外包上岸 🚢

索性辞职，全职学前端两个月。
年后通过外包，派遣到 上汽集团。

面试内容很简单：

  • 手写一个穿梭框
  • 我还记得当时用数组倒序删除解决问题

最终薪资 12k。
当时：

  • 我大专还没毕业
  • 旁边本科小姐姐 15k 😅

但已经很满足了。

疫情系统 & 政府项目漂流 🧳

上汽项目 3 个月结束后，开始各种派遣：

  • 苏州陆家镇政府（疫情系统，干了快一年）
  • 浙江湖州政府 / 国家电网（项目没上线就结束）
  • 苏州广电
  • 安徽马鞍山现代牧业（牛粪味 + 无偿加班地狱 🤢）

外包的通病：

拿你不当人，用到极限。

2022 ~ 2024：技术成长总结 🧠

技术认知转变

通过 QQ 群认识了一些大佬，
第一次听说：

  • 时间复杂度
  • 大 O

我还傻乎乎问过一句：

“大 O 是啥？”

现在想想挺好笑的 😂。

后来开始系统补基础：

  • 数据结构
  • C 语言
  • 算法（基础为主，滑窗 / DP 基本没坚持下来）

技术栈扩展

私下学过 / 用过的东西：

  • React
  • Electron
  • three.js
  • cesium
  • NestJS
  • zustand
  • plasmo
  • Rust（刷完了《Rust 圣经》🦀）
    https://course.rs/about-book.html

开源贡献

为NaiveUI提交了两个PR。当时是在现代牧业。 他们业务系统使用这个。有些组件用起来开发体验很差劲 / 产品体验差劲。 就了解怎么提交pr。fork了仓库，怀着忐忑的心情提交了pr。然后就被合并了。

https://github.com/tusen-ai/naive-ui/pull/5669 https://github.com/tusen-ai/naive-ui/pull/5582

2024：离开外包，稳定下来 🌱

从外包离职，进入现在这家公司。
开始大量写 React。

以前外包项目基本全是 Vue，
React 反而是在这家公司真正用熟的。

这一年很忙，博客几乎停更 ✍️。

2024 电子消费 🖥️

自己装了一台台式机（约 7000）：

  • AMD R7 7700
  • AMD 6750 GRE
  • 16G × 2 内存
  • 1T × 2 SSD

写到这里才发现，这一路确实走得歪歪扭扭。
但好像，每一步也都算数。 🌌

2025 年度总结：视野、技术与生活的一年

一、视野的变化 🌍

2025 年，我开始真正接触外网技术社区。

起点其实很随意—— 总有人说：想学到“真技术”，得去推特（X）看。

后来慢慢认识了一些技术朋友，发现他们几乎只使用 Telegram（纸飞机） 交流。 再之后，注册账号、进群、看讨论，也就成了很自然的事情。

科学上网大概是 2023 年左右开始的，时间已经有点模糊。 但可以确定的是： 上了X后, 从这一刻起，视野确实被打开了。

二、这一年的重要节点 ⭐

1. 参加 FEDAY 2025

这是一次非常有冲击力的线下经历。 第一次如此近距离地接触到一些长期仰慕的技术前辈和一线开发者。

你会很直观地意识到： 差距是真实存在的，但努力也是真实有方向的。

2. 技术圈子的变化

陆续加入了一些前端相关的技术群（主要在 Telegram）。

对比下来，整体技术密度确实比以往的 QQ 群高不少：

交流质量的变化，是我今年很明显的一个感受。

3. 成人本科正式注册

成人本科终于注册成功，选择的是 开放大学 · 计算机科学与技术。

选择理由很现实：

  • 难度可控
  • 成本可控
  • 更符合当前阶段的目标

名校成教的计科专业，课程和在校生基本一致，毕业压力不低。

三、技术成长记录 💻

这一年并没有“爆炸式突破”，但属于持续积累的一年。

1. TypeScript 类型体操

刷了一些类型体操题目，没有挑战地狱难度。 黄色难度刷完之后就暂时停了。

最深刻的，还是对条件类型的理解逐渐清晰：

T extends U ? X : Y

虽然不复杂，但第一次真正意识到类型系统的表达能力。

2. Rust 学习与实践

  • 完整刷完了 rustlings
  • 对 Rust 的所有权、生命周期有了更“工程化”的理解

随后使用 rs-pack 编写了一个 WebAssembly 开源库：

fast-md5-web

  • ⭐ 获得了 90+ Star
  • 🧩 实际使用者不多，目前只有 1 个 Issue

这件事让我第一次比较真实地感受到： Star ≠ 使用量，但依然值得记录。

3. 工程相关技术

  • 因业务需要，学习并使用了 napi-rs
  • 因 Markdown + 数学公式渲染的性能问题，开始接触并深入了解 unified 生态
  • 看到了好看的博客, Copy下来使用. 接触并尝试了 Astro

4. Tauri 与个人项目

出于对 Rust 的兴趣，学习并实践了 Tauri， 并尝试写了一个本地记账软件。

说实话：

  • 结构比较混乱
  • 功能偏技术验证
  • 产品思维基本为 0

但至少完成了一次 从 0 到 1 的桌面应用尝试。

四、电子产品更替 📱💻

  • MI 13 因频繁卡顿，换成了 iPhone 17
  • 入手 MacBook Air M4 · 16G

其实我很早就用过 Mac，只不过那是外包公司配的设备。 离职时需要归还，当时还在电脑上贴了不少贴纸。

那台是 M1 Pro + Touch Bar。 这次准备买新设备时才发现—— 那已经是 Touch Bar 的最后一代了 😂

五、生活碎片 🐱🍳

1. 养猫

今年养了一只猫，还给它准备了一个“大 House”。

2. 消费与日常

  • 看了不少 户晨风 的视频
  • 顺势开通了 山姆会员

山姆确实是一个容易让人“买着买着就超预算”的地方。

六、美团年报 🍚

整体来看，我并不算重度外卖用户。 更多时候还是自己做饭，或者直接去实体店吃。

七、其他学习记录 📚

  • 多邻国英语：13 分

以前英语基础非常差。 几乎没有系统学过，中专阶段也基本不教英语， 主要学的是物理、语文等课程。

现在至少已经迈出了第一步。 不快，但是真的在补。

结语 ✨

回看 2025 年，没有什么“人生转折点”。 但我能清楚地看到几件事：

  • 视野在变
  • 技术在积累

对我来说，今年的人生不算被荒废。

2026 年，希望能够保持这种—— 不焦虑、不躺平、慢慢走但不后退的状态。 共勉💪🏻
