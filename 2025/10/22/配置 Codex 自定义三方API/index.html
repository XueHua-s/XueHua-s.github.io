参数说明

几个关键配置:

  • model_provider: 设置成 "codex" 就行
  • model: 用的模型,这里是 gpt-4.1-mini
  • model_reasoning_effort: 推理强度,low/medium/high 三档,medium 够用
  • disable_response_storage: 不保存历史记录,开启更安全
  • base_url: API 地址,换成你自己的
  • env_key: 环境变量名,密钥别直接写配置文件里
